# 分配学生 - 下载模板说明

## 1. 404 错误（已处理）

**现象**：点击「分配学生」弹窗中的「下载模板」时请求  
`/course/course/{courseId}/assignStudents/importTemplate` 返回 404。

**原因**：后端未实现该接口（未提供「分配学生」专用导入模板下载）。

**当前处理**：前端改为请求已有接口 `course/student/importTemplate` 下载学生导入模板，作为分配学生导入的模板使用（学号等列格式与分配学生导入兼容），从而避免 404。

**若后端后续提供专用模板**：在 `src/views/course/course/index.vue` 的 `downloadAssignImportTemplate` 中，将  
`const templatePath = 'course/student/importTemplate'`  
改为  
`const templatePath = 'course/course/' + assignState.course.id + '/assignStudents/importTemplate'`  
并恢复对 `getAssignStudentsImportTemplatePath` 的导入与使用即可。

**后端建议**：如需专用模板，可新增接口：  
`POST /course/course/{courseId}/assignStudents/importTemplate`，返回与「分配学生导入」格式一致的 Excel 文件。

---

## 2. runtime.lastError（与项目无关）

**现象**：控制台出现  
`Unchecked runtime.lastError: Could not establish connection. Receiving end does not exist.`

**原因**：来自浏览器扩展（如 Chrome 扩展）与页面通信失败（例如扩展的 content script 未注入或已卸载），与本站前端/后端代码无关。

**处理**：可忽略；或禁用不用的扩展、使用无痕/无扩展窗口调试以排除干扰。
