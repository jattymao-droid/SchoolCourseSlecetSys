# 本地开发指南

## 启动后端（本地）

### 方法 1: 使用 IDE（推荐）

#### IntelliJ IDEA

1. **打开项目**
   - File → Open → 选择 `SchoolCourseSlecetSys` 目录
   - 等待 Maven 依赖下载完成

2. **配置数据库**
   - 确保本地 PostgreSQL 正在运行
   - 数据库名：`ry_vue`
   - 用户名：`postgres`
   - 密码：根据你的本地配置修改

3. **修改配置文件**（如果需要）
   - 打开 `ruoyi-admin/src/main/resources/application-druid.yml`
   - 修改数据库连接信息：
     ```yaml
     url: jdbc:postgresql://localhost:5432/ry_vue?useUnicode=true&characterEncoding=utf8
     username: postgres
     password: 你的密码
     ```

4. **启动 Redis**
   - 确保本地 Redis 正在运行（默认端口 6379）
   - Windows: 运行 `redis-server.exe`
   - Linux/Mac: `redis-server`

5. **运行主类**
   - 找到 `ruoyi-admin/src/main/java/com/ruoyi/RuoYiApplication.java`
   - 右键 → Run 'RuoYiApplication'
   - 或点击绿色运行按钮

6. **查看启动日志**
   - 看到 `Started RuoYiApplication in X.XXX seconds` 表示启动成功
   - 默认端口：8080
   - 访问：http://localhost:8080

#### VS Code

1. **安装扩展**
   - Extension Pack for Java
   - Spring Boot Extension Pack

2. **打开项目**
   - File → Open Folder → 选择 `SchoolCourseSlecetSys`

3. **配置数据库和 Redis**（同上）

4. **运行**
   - 打开 `RuoYiApplication.java`
   - 点击 `Run` 或 `Debug`

### 方法 2: 使用命令行

#### Windows PowerShell

```powershell
# 进入项目目录
cd SchoolCourseSlecetSys

# 确保已编译
mvn clean package -DskipTests

# 运行后端
java -jar ruoyi-admin/target/ruoyi-admin.jar
```

#### 使用 Maven 直接运行

```powershell
cd SchoolCourseSlecetSys/ruoyi-admin
mvn spring-boot:run
```

### 方法 3: 使用启动脚本

创建启动脚本 `start-local.ps1`:

```powershell
# 检查 PostgreSQL
$pgStatus = Get-Service -Name postgresql* -ErrorAction SilentlyContinue
if ($pgStatus.Status -ne 'Running') {
    Write-Host "PostgreSQL 未运行，请先启动 PostgreSQL" -ForegroundColor Red
    exit 1
}

# 检查 Redis
$redisProcess = Get-Process redis-server -ErrorAction SilentlyContinue
if (-not $redisProcess) {
    Write-Host "Redis 未运行，正在启动..." -ForegroundColor Yellow
    Start-Process redis-server -WindowStyle Minimized
    Start-Sleep -Seconds 2
}

# 启动后端
Write-Host "正在启动后端服务..." -ForegroundColor Green
cd SchoolCourseSlecetSys/ruoyi-admin
mvn spring-boot:run
```

运行：
```powershell
./start-local.ps1
```

## 本地开发环境要求

### 必需软件

1. **JDK 1.8+**
   - 下载：https://www.oracle.com/java/technologies/javase-downloads.html
   - 验证：`java -version`

2. **Maven 3.6+**
   - 下载：https://maven.apache.org/download.cgi
   - 验证：`mvn -version`

3. **Node.js 14+**
   - 下载：https://nodejs.org/
   - 验证：`node -version`

4. **PostgreSQL 12+**
   - 下载：https://www.postgresql.org/download/
   - 验证：`psql --version`

5. **Redis 6+**
   - Windows: https://github.com/microsoftarchive/redis/releases
   - 验证：`redis-cli ping` (应返回 PONG)

### 数据库初始化

```sql
-- 创建数据库
CREATE DATABASE ry_vue;

-- 导入数据（使用备份文件）
psql -U postgres -d ry_vue < sql/ry_vue_full_backup_20260308_210240.sql
```

## 常见问题

### 问题 1: 端口被占用

**错误**: `Port 8080 was already in use`

**解决**:
```powershell
# 查找占用 8080 端口的进程
netstat -ano | findstr :8080

# 结束进程（替换 PID）
taskkill /PID <PID> /F
```

### 问题 2: 数据库连接失败

**错误**: `Connection refused` 或 `Authentication failed`

**解决**:
1. 确认 PostgreSQL 正在运行
2. 检查数据库配置（用户名、密码、端口）
3. 确认数据库 `ry_vue` 已创建

### 问题 3: Redis 连接失败

**错误**: `Unable to connect to Redis`

**解决**:
1. 启动 Redis：`redis-server`
2. 测试连接：`redis-cli ping`

### 问题 4: Maven 依赖下载慢

**解决**: 配置国内镜像

编辑 `~/.m2/settings.xml`:
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

## 调试技巧

### 后端调试

1. **在 IDE 中设置断点**
   - 在代码行号左侧点击设置断点
   - 使用 Debug 模式启动

2. **查看日志**
   - 控制台输出
   - 日志文件：`logs/sys-info.log`

3. **修改日志级别**
   - 编辑 `application.yml`:
     ```yaml
     logging:
       level:
         com.ruoyi: debug
     ```

### 前端调试

1. **浏览器开发者工具**
   - F12 打开
   - Console 查看日志
   - Network 查看请求

2. **Vue DevTools**
   - Chrome 扩展
   - 查看组件状态

## 热重载

### 后端热重载

在 `application.yml` 中启用：
```yaml
spring:
  devtools:
    restart:
      enabled: true
```

### 前端热重载

`npm run dev` 自动支持热重载，修改代码后自动刷新。

**提示**: 本地开发时使用 `application.yml`（开发环境配置），生产部署时使用 `application-prod.yml`。
