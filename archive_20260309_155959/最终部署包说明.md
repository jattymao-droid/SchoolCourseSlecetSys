## 🆘 常见问题

### Q1: 部署脚本报错 "未找到 PostgreSQL Docker 容器"

**解决方案**:
```bash
# 检查容器是否运行
docker ps | grep postgres

# 如果容器未运行，启动容器
docker start <postgres_container_name>
```

### Q2: 数据库导入失败

**解决方案**:
```bash
# 检查容器内 PostgreSQL 状态
docker exec <postgres_container> psql -U postgres -c "SELECT 1"

# 手动导入数据库
docker cp sql/database_backup.sql <postgres_container>:/tmp/
docker exec -e PGPASSWORD=mm5621528 <postgres_container> psql -U postgres -d school-course -f /tmp/database_backup.sql
```

### Q3: 后端无法启动

**解决方案**:
```bash
# 查看后端日志
tail -100 /www/wwwroot/course-selection/logs/backend.log

# 检查 Java 版本
java -version

# 检查端口占用
netstat -tlnp | grep 8080
```

### Q4: 页面无法访问

**解决方案**:
```bash
# 检查 Nginx 状态
systemctl status nginx

# 检查域名解析
ping xk.xmls.vip

# 检查防火墙
firewall-cmd --list-ports
```
