## 🆘 遇到问题？

### 常见问题

1. **后端无法启动**
   - 查看日志: `tail -100 /www/wwwroot/course-selection/logs/backend.log`
   - 检查 Java 版本: `java -version`
   - 检查端口占用: `netstat -tlnp | grep 8080`

2. **数据库连接失败**
   - 检查 PostgreSQL 状态: `systemctl status postgresql`
   - 测试连接: `su - postgres -c "psql -d ry_vue"`

3. **页面无法访问**
   - 检查 Nginx 状态: `systemctl status nginx`
   - 检查域名解析: `ping xk.xmls.vip`
   - 检查防火墙: `firewall-cmd --list-ports`

4. **502 错误**
   - 检查后端是否运行: `bash /www/wwwroot/course-selection/scripts/status.sh`
   - 查看 Nginx 错误日志: `tail -100 /www/wwwlogs/xk.xmls.vip.error.log`

详细的故障排查步骤请参考部署包中的 `部署指南.md` 文档。
