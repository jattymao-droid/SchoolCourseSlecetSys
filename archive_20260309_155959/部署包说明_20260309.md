## 故障排查

### 后端启动失败

查看日志：
```bash
tail -f /www/wwwroot/course-selection/logs/backend.log
```

常见错误：
- 如果看到 `Could not resolve placeholder` 错误，说明 profile 配置有问题
- 如果看到数据库连接错误，检查 PostgreSQL 容器是否运行
- 如果看到 Redis 连接错误，检查 Redis 容器是否运行

### 接口 404 错误

1. 确认后端服务正在运行：
```bash
bash /www/wwwroot/course-selection/scripts/status.sh
```

2. 确认使用的是最新的 JAR 文件：
```bash
ls -lh /www/wwwroot/course-selection/backend/ruoyi-admin.jar
```

3. 重启后端服务：
```bash
bash /www/wwwroot/course-selection/scripts/restart-backend.sh
```

### Nginx 配置问题

测试 Nginx 配置：
```bash
nginx -t
```

重载 Nginx：
```bash
nginx -s reload
```
