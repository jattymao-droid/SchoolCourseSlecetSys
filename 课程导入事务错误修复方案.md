# 课程导入事务错误修复方案

## 问题分析

### 错误信息
```
ERROR: current transaction is aborted, commands ignored until end of transaction block
```

### 错误原因

这是PostgreSQL特有的事务处理机制导致的问题：

1. **事务中断机制**: 在PostgreSQL中，当事务内的某个SQL语句执行失败时，整个事务会进入"aborted"状态
2. **后续命令被拒绝**: 在事务aborted后，所有后续的SQL命令都会被拒绝执行，直到事务被回滚或提交
3. **异常捕获问题**: 代码中使用了 `try-catch` 捕获异常，但没有正确处理事务回滚，导致事务保持在aborted状态

### 具体场景

在 `importCourseWithQuota` 方法中：

```java
@Transactional(rollbackFor = Exception.class)
public String importCourseWithQuota(MultipartFile file, boolean updateSupport, String operName)
{
    // ...
    for (int rowIndex = 1; rowIndex <= sheet.getLastRowNum(); rowIndex++)
    {
        try
        {
            // 1. 这里可能发生异常（比如数据验证失败）
            SysGrade grade = findGradeByName(sheetName);
            
            // 2. 异常被捕获，但事务已经aborted
        }
        catch (Exception e)
        {
            // 3. 继续下一行，但事务仍然是aborted状态
            // 4. 下一次调用 findGradeByName 时，SQL被拒绝执行
            failureNum++;
            failureMsg.append(e.getMessage());
        }
    }
}
```

## 解决方案

### 方案 1: 修改事务传播行为（推荐）

将查询方法设置为 `REQUIRES_NEW` 或 `NOT_SUPPORTED`，使其在独立的事务中执行。

#### 修改 CourseServiceImpl.java

```java
// 在 findGradeByName 方法上添加注解
@Transactional(propagation = Propagation.NOT_SUPPORTED)
private SysGrade findGradeByName(String gradeName)
{
    SysGrade params = new SysGrade();
    params.setGradeName(gradeName);
    List<SysGrade> grades = gradeService.selectGradeList(params);
    for (SysGrade g : grades) {
        if (gradeName.equals(g.getGradeName())) {
            return g;
        }
    }
    return null;
}
```

### 方案 2: 预加载所有数据（更好的性能）

在导入开始前，一次性加载所有年级、学期、班级数据到内存中。

#### 修改 importCourseWithQuota 方法

```java
@Override
@Transactional(rollbackFor = Exception.class)
public String importCourseWithQuota(MultipartFile file, boolean updateSupport, String operName)
{
    if (file == null || file.isEmpty())
    {
        throw new ServiceException("上传文件不能为空");
    }
    
    try
    {
        // 预加载所有数据（在事务外执行）
        Map<String, SysGrade> gradeMap = preloadGrades();
        Map<String, SysSemester> semesterMap = preloadSemesters();
        Map<String, Map<String, SysClass>> classMap = preloadClasses();
        
        org.apache.poi.ss.usermodel.Workbook workbook = org.apache.poi.ss.usermodel.WorkbookFactory.create(file.getInputStream());
        
        int successNum = 0;
        int failureNum = 0;
        StringBuilder failureMsg = new StringBuilder();
        int maxFailureMsg = 10;
        
        // 遍历所有Sheet
        for (int sheetIndex = 0; sheetIndex < workbook.getNumberOfSheets(); sheetIndex++)
        {
            org.apache.poi.ss.usermodel.Sheet sheet = workbook.getSheetAt(sheetIndex);
            String sheetName = sheet.getSheetName();
            
            // 从预加载的Map中查找年级
            SysGrade grade = gradeMap.get(sheetName);
            if (grade == null)
            {
                failureMsg.append("<br/>Sheet「").append(sheetName).append("」对应的年级不存在，已跳过");
                continue;
            }
            
            // ... 其余代码使用预加载的数据
        }
        
        // ...
    }
    catch (Exception e)
    {
        throw new ServiceException("导入失败：" + e.getMessage());
    }
}

// 添加预加载方法
@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private Map<String, SysGrade> preloadGrades()
{
    List<SysGrade> grades = gradeService.selectGradeList(new SysGrade());
    Map<String, SysGrade> map = new HashMap<>();
    for (SysGrade grade : grades)
    {
        map.put(grade.getGradeName(), grade);
    }
    return map;
}

@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private Map<String, SysSemester> preloadSemesters()
{
    List<SysSemester> semesters = semesterService.selectSemesterList(new SysSemester());
    Map<String, SysSemester> map = new HashMap<>();
    for (SysSemester semester : semesters)
    {
        map.put(semester.getSemesterName(), semester);
    }
    return map;
}

@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private Map<String, Map<String, SysClass>> preloadClasses()
{
    List<SysClass> classes = classMapper.selectClassList(new SysClass());
    Map<String, Map<String, SysClass>> map = new HashMap<>();
    for (SysClass clazz : classes)
    {
        String gradeName = clazz.getGradeName();
        if (!map.containsKey(gradeName))
        {
            map.put(gradeName, new HashMap<>());
        }
        map.get(gradeName).put(clazz.getClassName(), clazz);
    }
    return map;
}
```

### 方案 3: 临时快速修复（不推荐，但可以快速解决问题）

在每次捕获异常后，手动标记事务为rollback-only，然后在外层重新开启新事务。

### 完整的修改代码

```java

// 在 CourseServiceImpl 类中添加/修改以下方法：

@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private SysGrade findGradeByName(String gradeName)

    for (SysGrade g : grades) {
        if (gradeName.equals(g.getGradeName())) {
            return g;
        }
    }
    return null;
}

@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private SysSemester findSemesterByName(String semesterName)
{
    SysSemester params = new SysSemester();
    params.setSemesterName(semesterName);
    List<SysSemester> semesters = semesterService.selectSemesterList(params);
    for (SysSemester s : semesters) {
        if (semesterName.equals(s.getSemesterName())) {
            return s;
        }
    }
    return null;
}

@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private SysClass findClassByGradeAndShortName(String gradeName, String shortName)
{
    SysClass c = classMapper.selectByGradeNameAndClassName(gradeName, shortName);
    if (c != null) {
        return c;
    }
    String fullName = gradeName + shortName;
    return classMapper.selectByGradeNameAndClassName(gradeName, fullName);
}

// findOrCreateTeacherUser 方法需要特殊处理，因为它可能需要创建用户
// 将查询部分和创建部分分开
@Transactional(propagation = Propagation.NOT_SUPPORTED, readOnly = true)
private SysUser findTeacherUser(String teacherName)
{
    // 1. 按昵称精确查找
    SysUser user = findUserByNickName(teacherName);
    if (user != null) {
        return user;
    }
    // 2. 按 userName=teacher_姓名 查找
    String userName = "teacher_" + teacherName;
    user = userService.selectUserByUserName(userName);
    return user;
}

@Transactional(propagation = Propagation.REQUIRES_NEW, rollbackFor = Exception.class)
private SysUser createTeacherUser(String teacherName, String operName)
{
    String userName = "teacher_" + teacherName;
    SysUser user = buildTeacherUser(teacherName);
    user.setCreateBy(operName);
    
    // 检查部门是否存在
    SysDept dept = deptService.selectDeptById(DEFAULT_DEPT_ID);
    if (dept != null) {
        user.setDeptId(DEFAULT_DEPT_ID);
    } else {
        SysDept queryDept = new SysDept();
        queryDept.setStatus("0");
        List<SysDept> deptList = deptService.selectDeptList(queryDept);
        if (!deptList.isEmpty()) {
            user.setDeptId(deptList.get(0).getDeptId());
        } else {
            throw new ServiceException("无法自动创建教师用户：未找到可用部门");
        }
    }

    try {
        userService.insertUser(user);
        if (user.getUserId() == null) {
            user = userService.selectUserByUserName(userName);
        }
    } catch (Exception e) {
        user = userService.selectUserByUserName(userName);
        if (user == null) {
            throw new ServiceException("创建教师用户失败：" + e.getMessage());
        }
    }
    return user;
}

// 修改原来的 findOrCreateTeacherUser 方法
private SysUser findOrCreateTeacherUser(String teacherName, String operName)
{
    // 先查找
    SysUser user = findTeacherUser(teacherName);
    if (user != null) {
        return user;
    }
    // 不存在则创建（在新事务中）
    return createTeacherUser(teacherName, operName);
}
```

## 注意事项

1. **事务传播行为**: `Propagation.NOT_SUPPORTED` 表示以非事务方式执行，如果当前存在事务，则挂起当前事务
2. **只读事务**: 对于纯查询方法，添加 `readOnly = true` 可以提高性能
3. **创建操作**: `findOrCreateTeacherUser` 中的创建部分使用 `Propagation.REQUIRES_NEW`，确保在新事务中执行
4. **性能考虑**: 如果导入数据量很大，建议使用方案2（预加载数据）
