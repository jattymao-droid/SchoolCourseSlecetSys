# 学校课程选课系统

## 项目简介

基于若依框架的学校课程选课管理系统，支持学生选课、教师管理、课程管理等功能。

## 技术栈

- **后端**: Spring Boot 2.5.15 + MyBatis + PostgreSQL
- **前端**: Vue 2 + Element UI
- **移动端**: uni-app
- **缓存**: Redis
- **构建工具**: Maven + npm

## 快速启动

### 本地开发环境

1. **启动所有服务**（推荐）
   ```powershell
   .\start-all.ps1
   ```


## 注意事项

1. **Redis 必须运行**: 系统依赖 Redis 缓存，启动前确保 Redis 已运行
2. **数据库连接**: 确保 PostgreSQL 数据库已创建并可连接
3. **端口占用**: 确保 8080（后端）和 80（前端）端口未被占用
4. **学生删除**: 系统使用物理删除，删除学生后可重新导入相同学号

## 常见问题

### 1. 后端启动失败

检查：
- PostgreSQL 是否运行
- Redis 是否运行
- 数据库配置是否正确
- 端口 8080 是否被占用

### 2. 前端无法访问后端

检查：
- 后端是否成功启动
- 前端配置的 API 地址是否正确（`.env.development`）

### 3. 学生导入失败

常见原因：
- Excel 格式不正确
- 年级或班级不存在
- 学号已存在
