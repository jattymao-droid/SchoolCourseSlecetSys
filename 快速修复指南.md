## 🔧 立即执行的 3 个步骤

### 步骤 1：修复 Nginx 配置（5 分钟）

1. 打开宝塔面板 -> 网站 -> xk.xmls.vip -> 设置 -> 配置文件
2. **全选删除**现有配置
3. 复制粘贴以下配置：

```nginx
server {
    listen 80;
    server_name xk.xmls.vip;
    index index.html;
    root /www/wwwroot/xk.xmls.vip/frontend;
    
    location /prod-api/ {
        proxy_pass http://127.0.0.1:8080/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    access_log /www/wwwlogs/xk.xmls.vip.log;
    error_log /www/wwwlogs/xk.xmls.vip.error.log;
}
```

4. 点击"保存"
5. 在 SSH 中执行：`nginx -s reload`

---

### 步骤 2：上传新的后端 JAR 包（5 分钟）

**本地文件位置**：`SchoolCourseSlecetSys/deploy/backend/ruoyi-admin.jar`

**上传到服务器**：

```bash

# 停止后端服务

# 上传到：/www/wwwroot/xk.xmls.vip/backend/ruoyi-admin.jar
```
