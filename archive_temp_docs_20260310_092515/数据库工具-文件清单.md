## 🚀 上传到服务器

### 方法 1: 单独上传（推荐用于首次使用）

```bash
# 上传核心工具
scp diagnose-database-server.sh root@your-server:/root/
scp auto-fix-database-server.sh root@your-server:/root/
scp manage-database-server.sh root@your-server:/root/

# 上传核心文档
scp 数据库工具-README.md root@your-server:/root/
scp 数据库工具-快速参考.md root@your-server:/root/
```

### 方法 2: 打包上传（推荐用于完整安装）

```bash
# 本地打包
chmod +x package-database-tools.sh
./package-database-tools.sh

# 上传压缩包
scp database-tools-*.tar.gz root@your-server:/root/

# 服务器上解压
ssh root@your-server
cd /root
tar -xzf database-tools-*.tar.gz
cd database-tools-*
chmod +x *.sh
```

### 方法 3: 使用宝塔面板

1. 登录宝塔面板
2. 进入文件管理
3. 上传所有 .sh 和 .md 文件到 /root 目录
4. 在终端中执行：`chmod +x /root/*.sh`
