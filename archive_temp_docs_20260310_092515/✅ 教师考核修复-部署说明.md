## 🔍 故障排查

### 问题 1：后端启动失败

**检查**：
```bash
# 查看完整日志
cat /www/wwwroot/xk.xmls.vip/backend/backend.log

# 检查 JAR 文件
ls -lh /www/wwwroot/xk.xmls.vip/backend/ruoyi-admin.jar
```

**常见原因**：
- JAR 文件上传不完整（大小不对）
- 端口被占用
- 数据库连接失败

### 问题 2：平均得分仍然显示 0

**检查**：
1. 确认 JAR 文件已更新（查看文件时间戳）
2. 确认后端已重启（查看日志中的启动时间）
3. 确认数据库中有评价数据：
   ```bash
   docker exec postgresql_d6wx-postgresql_D6Wx-1 psql -U postgres -d school-course -c "SELECT COUNT(*) FROM stu_evaluation WHERE semester_id = 学期ID;"
   ```

### 问题 3：评价详情页面为空

**检查**：
1. 浏览器控制台是否有 API 错误（F12 → Console）
2. 后端日志是否有错误
3. 数据库中是否有该教师的评价数据：
   ```bash
   docker exec postgresql_d6wx-postgresql_D6Wx-1 psql -U postgres -d school-course -c "SELECT COUNT(*) FROM stu_evaluation e JOIN cou_course c ON e.course_id = c.id WHERE c.teacher_id = 教师ID;"
   ```

### 问题 4：页面显示异常

**解决**：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 强制刷新页面（Ctrl+F5）
3. 检查前端是否需要重新编译
