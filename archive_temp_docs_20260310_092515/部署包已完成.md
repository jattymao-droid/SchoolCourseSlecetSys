## 🚀 快速部署步骤

### 1️⃣ 上传到服务器

使用以下任一方式上传 `deploy` 目录到服务器：

**方式A：使用SCP命令**
```bash
scp -r deploy root@your-server-ip:/opt/
```

**方式B：使用宝塔面板**
- 登录宝塔面板
- 进入文件管理
- 上传 `deploy` 目录到 `/opt/`

### 2️⃣ 执行一键部署

SSH登录到服务器后执行：

```bash
cd /opt/deploy/scripts
chmod +x deploy.sh
./deploy.sh
```

脚本会自动完成：
- ✅ 检查环境（Java、Docker、PostgreSQL、Redis）
- ✅ 创建部署目录 `/opt/school-course`
- ✅ 复制所有文件
- ✅ 创建数据库 `school-course`
- ✅ 导入数据库
- ✅ 启动后端服务（端口8080）
- ✅ 生成Nginx配置

### 3️⃣ 配置Nginx（宝塔面板）

在宝塔面板中：

1. **创建网站**
   - 域名：你的域名或服务器IP
   - 根目录：`/opt/school-course/frontend`
   - PHP版本：纯静态

2. **配置反向代理**
   - 代理名称：`backend-api`
   - 目标URL：`http://127.0.0.1:8080`
   - 发送域名：`$host`
   - 代理路径：`/prod-api`

3. **保存并重启Nginx**

### 4️⃣ 访问系统

- 前端地址：`http://your-server-ip`
- 默认账号：`admin`
- 默认密码：`admin123`

## 🔍 故障排查

### 后端无法启动
```bash
# 检查Java版本
java -version

# 检查端口占用
netstat -tlnp | grep 8080

# 查看详细日志
tail -f /opt/school-course/backend/backend.log
```

### 数据库连接失败
```bash
# 检查PostgreSQL容器
docker ps | grep postgres

# 测试数据库连接
docker exec -it $(docker ps | grep postgres | awk '{print $1}') psql -U postgres -d school-course
```

### Redis连接失败
```bash
# 检查Redis容器
docker ps | grep redis

# 测试Redis连接
docker exec -it $(docker ps | grep redis | awk '{print $1}') redis-cli ping
```
