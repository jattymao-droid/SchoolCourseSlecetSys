## 🐛 故障排查

### 问题 1：无法访问网站

**检查**：
```bash
# 检查 Nginx 状态
systemctl status nginx

# 检查端口
netstat -tlnp | grep 80
```

### 问题 2：API 请求失败（502）

**检查**：
```bash
# 检查后端服务
cd /www/wwwroot/xk.xmls.vip/backend
./status.sh

# 查看后端日志
tail -f backend.log
```

### 问题 3：数据库连接失败

**检查**：
```bash
# 检查 PostgreSQL 容器
docker ps | grep postgresql

# 测试数据库连接
docker exec postgresql_d6wx-postgresql_D6Wx-1 psql -U postgres -c "SELECT 1"
```

### 问题 4：Redis 连接失败

**检查**：
```bash
# 检查 Redis 服务
systemctl status redis

# 测试 Redis 连接
redis-cli ping
```
